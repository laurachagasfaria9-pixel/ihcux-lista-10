# Breve descrição da ideia:
Criar poucas telas e abas com informações intuitivas para o usuário
# Explicação das telas criadas e fluxo
Criamos seis telas, mostrando o seguinte:
1) opções de criar conta ou logar com dados cadastrados;
2) tela para criar perfil
3) escolher um estabelecimento para visualisar (através de duas opções: carrossel e seleção por lista)
4) breve descrição do estabelecimento escolhido e um link para ir para sua fila
5) escolher data e horário que pretende ir ao local
6) apresentação da fila já com a sua posição, quantidade de pessoas e tempo estimado

Além disso, criamos duas abas de direcionamento para caso a pessoa queira alterar o perfil ou também cancelar reserva do dia.
